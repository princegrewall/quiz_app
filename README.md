# **Quiz Application**

A modern web-based Quiz Application built with **React, TypeScript, Tailwind CSS, shadcn-ui**, and a **Node.js/Express** backend with **MongoDB**. Users can take quizzes, track scores, and submit results.  

---

## **ðŸš€ Features**

- Fetches quiz questions dynamically from **Open Trivia Database (OpenTDB)**  
- Multiple-choice quizzes with question shuffling  
- Tracks visited questions and user answers  
- Submit quiz results with optional email  
- View recent submissions (for admins/debugging)  
- Fully responsive frontend  

---

## **Tech Stack**

**Frontend:**

- React 18 + TypeScript  
- Vite  
- Tailwind CSS  
- shadcn-ui components  

**Backend:**

- Node.js + Express  
- MongoDB with Mongoose
