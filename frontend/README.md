Project Setup

This project is built using React + TypeScript + Vite with Tailwind CSS and shadcn-ui.

Tech Stack

React

TypeScript

Vite

Tailwind CSS

shadcn-ui

How to Run the Project Locally
1️⃣ Clone the repository
git clone <YOUR_GIT_URL>

2️⃣ Go to the project folder
cd <PROJECT_NAME>

3️⃣ Install dependencies
npm install

4️⃣ Start the development server
npm run dev


The app will run on:

http://localhost:5173

Project Structure (Frontend)
src/
 ├── components/
 ├── pages/
 ├── hooks/
 ├── lib/
 ├── index.css
 └── main.tsx

Build for Production
npm run build

Notes

Make sure Node.js and npm are installed.

Update environment variables if required before running the project.
